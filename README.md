# observational-astronomy-foundation

Observational astronomy laboratory reports and data analysis projects.
Developed for foundational astronomy coursework at Stony Brook University.

### Spectroscopy
- **Spectral Line Identification** - Classification of atomic and molecular features
- **Wavelength Calibration** - Arc lamp calibration and dispersion solutions
- **Radial Velocity Measurement** - Doppler shift calculations
- **Equivalent Width Analysis** - Line strength quantification
- **Uncertainty Propagation** - Measurement error analysis

### Exoplanet Transit
- **Light Curve Reduction** - Photometric calibration and normalization
- **Transit Modeling** - Depth, duration, and ingress/egress analysis
- **Orbital Period Determination** - Period fitting from time-series data
- **Planetary Radius Estimation** - Radius extraction from transit depth
- **Error Analysis** - Statistical and systematic uncertainty estimation

### CCD Instrument Calibration
- **Bias Frame Analysis** - Read noise characterization
- **Dark Current Measurement** - Thermal noise evaluation
- **Flat-Field Correction** - Pixel response calibration
- **Gain Determination** - ADU-to-electron conversion factor
- **Linearity Testing** - Detector response verification
